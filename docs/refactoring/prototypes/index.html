🎨 Спецификации дизайн-системы

Цветовая палитра

Primary: #3B82F6, Success: #10B981, Warning: #F59E0B, Error: #EF4444. Полная шкала от 50 до 900.

Типографика

Inter для UI (400-700), JetBrains Mono для кода. 6 уровней заголовков, 3 размера body текста.

Spacing System

4px базовая единица. Шкала от 4px до 96px. Консистентные отступы и промежутки.

Компоненты

4 типа кнопок, 5 состояний input'ов, 3 варианта карточек. Все с hover состояниями.

Shadows & Radius

3 уровня теней, border-radius от 4px до 24px. Современный и чистый вид.

Responsive Design

Mobile-first подход. Breakpoints: 375px, 768px, 1024px, 1440px.
